# Book Tracker

## Description
"Book Tracker" — GUI-приложение для управления списком прочитанных книг. Позволяет добавлять книги, фильтровать по жанру и количеству страниц, сохранять данные в JSON и загружать их при запуске.

## Инструкция по запуску
1. Установите Python 3.x.
2. Скачайте проект и откройте его в редакторе.
3. Запустите файл `main.py`.
4. Введите информацию о книге и нажмите "Добавить книгу".
5. Используйте фильтры для поиска по жанру или количеству страниц.
6. Данные сохраняются автоматически при добавлении.

## Примеры использования
- Добавление новых книг.
- Фильтрация по жанру "Фантастика".
- Фильтрация по страницам больше 200.
